# AiT Agency — Agent Rules

## Project
AI Workers service backend. Next.js 14 + Supabase + Stripe + Clerk.

## Stack
- Next.js 14 App Router, TypeScript strict
- Tailwind CSS, Geist font
- Supabase for leads + client data
- Stripe for Blueprint/Build/Managed billing
- Clerk for client portal auth
- Slack webhooks for intake notifications
